    random-color 

    @function random-color($multiplier: 1, 1, 1, $opacity: 1) { ... }

    Description

    Generates a random color. By default, it is unbiased, and the colors will almost certainly be translucent. Optionally, you can pass a list of multipliers to weigh the output. For instance, if you pass a red modifier of 2, the function will multiply the maximum red value, 255, by 2, getting 510, find a random number between 1 and 510, and, finally, return that number as the R value, while clamping any number over 255 to 255. This means that, with a $red multiplier of 2, there is a ~50% chance of the red value being 255; a ~67% chance that a $red value of 3 will result in a red value of 255; and so on. To generate a hex value instead of an RGBa value, pass an $opacity value of 1.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $multiplier

    Red, green, and blue multipliers

    		 List 1, 1, 1
    $opacity

    Whether the output should be opaque

    		 Number 1

    Example

    // SCSS
span {
 color: random-color(3, 1, 1);
}
    // CSS
span {
  color: rgba(229, 240, 108);
}

    Throws

    • RGBa multipliers must be positive.

    random-color-by-hue 

    @function random-color-by-hue($hue, $opacity: 1) { ... }

    Description

    Given a hue value, generates a color with random saturation and lightness, and, if the opacity parameter default of 1 is overriden, random opacity. To increase the chances of a somewhat-expected color being output, saturation is clamped between 20 and 100, and lightness between 20 and 80.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $hue

    A hue value between 0 and 360

    		 Number — none
    $opacity

    The opacity of the generated color

    		 Number 1

    Example

    // SCSS
span {
 color: random-color-by-hue(230);
}
    // CSS
span {
  color: #1d35af;
}

    random-mix 

    @function random-mix($color-1, $color-2, $min-1: 10%, $max-1: 90%) { ... }

    Description

    Given two colors, mix them, randomizing how much of each color to use in so doing. Optionally, pass in the lowest and highest amount of the first color that will be used in generating the mixture.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $color-1

    The first color to be used in generating a mixed color

    $color-2

    The second color to be used in generating a mixed color

    		 Color — none
    $min-1

    The lowest amount of the first color that will be used in mixing

    		 Number 10%
    $max-1

    The highest amount of the first color that will be used in mixing

    		 Number 90%

    Example

    // SCSS

 background: random-mix(red, blue);

    random-animate 

    @mixin random-animate($num-elements, $prop-to-animate, $animation-props, $steps, $function-name, $args...) { ... }

    Description

    Randomly animates a number of nth-children.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $num-elements

    The number of elements to animate

    		 Number — none
    $prop-to-animate

    The property of those elements which should be animated

    		 String — none
    $animation-props

    The animation properties to use for the animation

    		 List — none
    $steps

    The number of times each element will be animated

    		 Number — none
    $function-name

    The randomizing function to be called for use in the animation

    		 String — none
    $args...

    Parameters to be passed to the $function-name function

    		 String — none

    Example

     color: rgba(202, 105, 191, 0.7);
   }
   100% {
     color: rgba(59, 238, 72, 0.7);
   }
 }

   }

    random-iterate 

    @mixin random-iterate($number-of-items, $property, $function, $args...) { ... }

    Description

    Given a number of elements, a property, and a function to call upon those elements, call the function separateley on each property of each element; like random-animate, but without the whole 'animation' part.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $number-of-items

    The number of elements to be iterated over

    		 Number — none
    $property

    The property of those elements which will call the function

    		 String — none
    $function

    The function to be called on each element

    		 String — none
    $args...

    Parameters, if any, to be passed to that function

    		 List — none

    Example

    // SCSS
span {
  color: random-iterate(3, color, random-hex);
}
    // CSS
span:nth-child(3) {
  color: #488176;
}
span:nth-child(2) {
  color: #0C0FC3;
}
span:nth-child(1) {
  color: #DB47C4;
}

    Throws

    • $number-of-items in random-iterate must be an integer greater than or equal to 1
